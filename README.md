# Play Change Foundation

A Next.js website for the Play Change Foundation organization.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js App Router pages
- `components/` - React components
- `public/` - Static assets
- `images/` - Image assets
- `css/` - Stylesheets
- `js/` - JavaScript files

## Features

- Responsive design
- About page
- Contact form
- Initiatives page
- SEO optimization with robots.txt and sitemap.xml
